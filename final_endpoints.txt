GET /api/final-posting: Get All Posting | Get all final posting members with pagination.
POST /api/final-posting: Create Posting | Create a new final posting member.
GET /api/final-posting/{posting_id}: Get Posting By Id | Get a final posting member by ID.
PUT /api/final-posting/{posting_id}: Update Posting | Update an existing final posting member.
DELETE /api/final-posting/{posting_id}: Delete Posting | Delete a final posting member.
DELETE /api/final-posting/all: Delete All Posting | Delete all final posting records. Use with caution!
POST /api/final-posting/bulk-delete: Bulk Delete Posting | Delete multiple final posting records by IDs.
POST /api/final-posting/archive: Archive Current Postings | Archive all records from the current Posting table to the FinalPosting table.
This copies all data but does not delete from the original table.
